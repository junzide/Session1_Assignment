# Simple Student Grades Management System

---

> 本项目使用C++20进行编写，设计

## C++20特性使用

1. 使用了C++17的`std::filesystem`
2. 使用了`std::format`的标准库
3. 使用了`std::string_view`

​	...等特性

## 功能实现

1. 读取在可执行文件exe目录下的assets文件夹中的各种老师和学生的账户与信息
2. 对存储的信息进行一定限度的查看，修改和保存
3. 显示老师的信息
4. 显示学生的信息
5. 对学生的总分成绩进行排序
6. 增加学生的数据
7. 删除学生的数据
8. 查找学生
9. 修改学生的信息并保存
